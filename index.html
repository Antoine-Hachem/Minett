Building charts, palettes, and patterns

Block palette templates
  • Rustic:Spruce + Stone + Cobble + Dark Oak accents + Moss
  • Medieval:Oak + Dark Oak + Stone Brick + Deep Slate trim + Lanterns
  • Modern:Quartz + Concrete (white/gray) + Glass + Sea Lanterns
  • Desert:Sandstone + Smooth Sandstone + Acacia + Terracotta
  • Mountain:Spruce + Packed/Polished Ice accents + Basalt + Deep Slate
Pro patterns
  • Depth:Alternate walls: 1 block inset + stair/half-slab trim + window sills
  • Roofing:A-frame (stairs), dormers (fences + slabs), overhang 1–2 blocks
  • Framing:Log beams on corners + crossbeams every 5–7 blocks
  • Texture:Mix 3 blocks: base (70%), detail (20%), highlight (10%)
  • Lighting:Hidden light: sea lantern under carpet, glow lichen + bone meal
Common pitfalls
  • Flat walls:Break planes with stairs/slabs, flower boxes, supports
  • Monotone:Use gradient: cobble → stone → stone brick → deepslate
  • Scale:Door/window size proportional to build; keep module 5–9 blocks
  • Over-detail:Leave negative space; detail focal points only

Redstone circuits and practical builds

Core components
  • Power:Redstone dust, blocks, torches, repeaters, comparators
  • Movement:Pistons, sticky pistons, observers, droppers, dispensers
  • Logic:Torch-based NOT, repeater locks, comparator subtraction
Blueprints
  • Item sorter:Comparator → 3 dust → repeater → locking torch; 41 filters
  • Door (3×3):Double piston extenders with observer timing
  • Mob farm:Flush with water or trident killer (Bedrock)
  • Sugar cane:Observers per stalk; hopper minecart collection
  • Super smelter:Minecart loaders with furnace arrays (8–32 furnaces)
Timing tips
  • Repeater delay:1–4 ticks; chain for longer pulses
  • Observer clocks:2-tick pulses; buffer with repeaters to stabilize
  • Chunk borders:Keep contraptions within chunks; add start-up reset

Survival progression and combat

  • Iron age:Iron tools, portal, enchanting table, nether access
  • Diamond age:Fortune III, mending, villager trading hall

Combat essentials
  • Loadout:Shield, bow (Power V), sword (Sharpness V), totem
  • Potions:Strength II, Regeneration, Swiftness, Fire Resistance

Hardcore habits

Visual tutorials for automatic farms

From simple early-game crop automation to reliable mid-game XP grinders and scalable late-game iron factories. Each card includes a visual sketch and build notes.

Early-game — crop auto-harvest
  • Design:Observer detects growth → piston breaks → hopper minecart collects
  • Blocks:Observer, piston, rails, hopper, chest, water, farmland
  • Footprint:Compact 5×7; scalable in modules
  • Tip:Use daylight sensor or clock to batch-harvest for fewer entity updates
Mid-game — spawner XP grinder
  • Design:Spawner room (9×9) → water streams → drop chute (22–24 blocks) → kill chamber
  • Blocks:Spawner, slabs, signs/trapdoors, water, hoppers, chests
  • Footprint:Room 9×9×6; vertical chute beneath
  • Tip:AFK point within 16 blocks of spawner; lightproof the room
Late-game — iron golem factory
  • Design:Villager pods + zombie scare → golem spawn platform → water push → lava blade collection
  • Blocks:Beds, workstations, villagers, zombie, trapdoors, water, lava, hoppers
  • Footprint:Modular 3-pod units; stack vertically or side-by-side
  • Tip:Chunk-align modules; add start-up reset and day/night handling

Enchanting and gear progression

Table setup
  • Max power:15 bookshelves; lapis reserve chest
  • Rerolls:Cycle with wooden tools to rotate enchant pool
  • Anvil work:Combine lowest cost first; track prior work penalty
Best-in-slot
  • Tools:Efficiency V, Unbreaking III, Mending; Fortune/Silk Touch alternates
  • Combat:Protection IV, Feather Falling IV, Thorns optional
  • Utility:Depth Strider III, Respiration III, Frost Walker situational
XP & costs
  • XP farms:Ender, blaze, or spawner-based; bottle storage
  • Books:Villager librarians, fishing early-game
  • Repair:Mending on all; avoid too many combines

Nether and End progression

Nether goals
  • Debris:Ancient debris at Y 15; TNT strip mines or beds (Nether)
  • Bartering:Gold farm → piglin barters for obsidian/pearls/blackstone
  • Fortress:Blaze rods for brewing; wither skeleton farm later
End goals
  • Dragon:Beds or bow; pillar climbs with water
  • Elytra:End city raids with shulker box prep
  • Chorus:Teleportation utility; farm for decorative blocks
Navigation & safety
  • Portal links:Overworld coords ÷ 8; build portals at exact nether coords
  • Bridging:Shift + slab edges; fire resistance active
  • Void risk:Slow falling + ender pearls; carry extra blocks
